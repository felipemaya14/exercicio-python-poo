Crie um repositório no GitHub chamado exercicio-python-poo.

Crie um arquivo chamado conta.py.

Programe uma classe chamada ContaBancaria que tenha:

Atributos: titular (nome da pessoa) e saldo (começa com 0).

Método depositar(valor): Adiciona o valor ao saldo.

Método sacar(valor): Verifica se há saldo suficiente. Se houver, desconta o valor; se não, exibe uma mensagem de "Saldo insuficiente".

No final do arquivo, crie uma instância dessa conta, faça um depósito e um saque para testar.

Suba o código para o GitHub e cole o link do repositório aqui.
